# -----------------------------------------------------------
# FRAMEWORK REFERENCE (FOR AI USE)
# -----------------------------------------------------------
#
# Use this reference for framework selection when RCP-076
# report is not available. These are training-data patterns,
# not evidence-based rankings.
#
# AIDA: Attention, Interest, Desire, Action
#   Best for: Promotional content, ads, sales pages, emails
#   Needs: Enough space for 4 distinct sections
#   Strength: Well-established, versatile
#   Weakness: Can feel formulaic for sophisticated audiences
#
# PAS: Problem, Agitate, Solution
#   Best for: Problem-solving products, pain point marketing
#   Needs: Clear audience pain point
#   Strength: Emotionally powerful, works at any length
#   Weakness: Can feel manipulative if agitation is overdone
#
# BAB: Before, After, Bridge
#   Best for: Transformation stories, coaching, education
#   Needs: Clear before/after contrast
#   Strength: Narrative-driven, builds aspiration
#   Weakness: Requires believable transformation claim
#
# FAB: Features, Advantages, Benefits
#   Best for: Product descriptions, technical products
#   Needs: Concrete features to translate into benefits
#   Strength: Logical progression, builds value
#   Weakness: Can feel dry without emotional layer
#
# ACC: Awareness, Comprehension, Conviction
#   Best for: Educational content, awareness campaigns
#   Needs: Topic the audience may not be aware of
#   Strength: Builds understanding before asking for action
#   Weakness: Slower to reach the call-to-action
#
# 4Ps: Promise, Picture, Proof, Push
#   Best for: Direct response, email sequences, launches
#   Needs: Strong proof elements (testimonials, data)
#   Strength: Bold and action-driving
#   Weakness: Promise must be deliverable; requires proof
#
# 4Cs: Clear, Concise, Compelling, Credible
#   Best for: Quality optimization (more checklist than
#   structure), any content type
#   Needs: Existing content to optimize
#   Strength: Universally applicable quality check
#   Weakness: Not a structural framework — works as overlay
#
# 4Us: Useful, Urgent, Unique, Ultra-specific
#   Best for: Headlines, subject lines, hooks, short-form
#   Needs: Very little space; best for single lines
#   Strength: Excellent for short-form optimization
#   Weakness: Not suitable for long-form structure
#
# QUEST: Qualify, Understand, Educate, Stimulate, Transition
#   Best for: Long-form sales, webinars, high-ticket offers
#   Needs: Space for 5 sections; complex value proposition
#   Strength: Thorough, builds deep reader engagement
#   Weakness: Too long for short-form content
#
# SLAP: Stop, Look, Act, Purchase
#   Best for: Social media ads, quick-scroll content
